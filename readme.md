## Quick Start

install python > 3.9

```
pip install rximg
python -m rximg.app
```
open 127.0.0.1:5000 in browser


## Develop 

install python > 3.9

download release
```
pip install -r requirements.txt
python app.py
```
open another terminal

```
cd frontend
npm run serve
```
open 127.0.0.1:8080 in browser
